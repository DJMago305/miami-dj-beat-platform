0. Introdução Geral – MDJPRO

👁️ NOTE
**AVISO IMPORTANTE:** O MDJPRO trabalha diretamente com ficheiros e estruturas reais do sistema. Embora o software tenha sido concebido com protocolos de segurança, **a responsabilidade final pela utilização correta recai sobre o utilizador.** Recomenda-se sempre trabalhar com cópias de segurança.

Introdução

MDJPRO (Magic DJ Pro) é uma plataforma profissional de gestão, auditoria e organização musical projetada para DJs, produtores e operadores de eventos que trabalham sob os padrões reais da indústria.

Desenvolvido pela Miami DJ Beat, o MDJPRO não é um reprodutor, nem um simples organizador de ficheiros. É um sistema de controlo de biblioteca musical criado para proteger, estruturar e otimizar um dos ativos mais valiosos de um DJ profissional: a sua base de dados musical.

Num ambiente onde erros de organização, metadados corrompidos ou digitalizações incorretas podem comprometer uma performance ao vivo, o MDJPRO atua como um firewall operacional entre o caos do disco rígido e a estabilidade exigida por softwares como Serato DJ, Rekordbox e VirtualDJ.

Filosofia do Sistema

O MDJPRO foi construído sobre um princípio fundamental:

"A música profissional não se improvisa. Estrutura-se, valida-se e protege-se."

Por esta razão, a aplicação opera através de fluxos de trabalho controlados, regras claras e validações constantes. Cada ação dentro do sistema — desde o carregamento de uma pasta raiz até à edição de um metadado — responde a uma lógica concebida para prevenir erros humanos, perda de dados e desorganização progressiva.

O MDJPRO não substitui o software de DJ. Prepara-o, alimenta-o corretamente e mantém-no estável.

Para quem é o MDJPRO?

  • DJs profissionais e residentes
  • DJs de eventos móveis e especiais
  • Produtores e curadores musicais
  • Academias e escolas de DJ
  • Empresas de entretenimento e eventos
  • Utilizadores que trabalham com bibliotecas grandes e multi-dispositivo

Se a sua música faz parte da sua identidade profissional e do seu sostento, o MDJPRO foi projetado para si.

O que o MDJPRO Resolve

  • Bibliotecas desordenadas e difíceis de manter
  • Metadados incompletos, falsos ou inconsistentes
  • Erros de digitalização no Serato ou noutros softwares
  • Perda de caminhos ao mover discos ou pastas
  • Falta de estrutura para eventos, sets e décadas
  • Risco de danificar a base de dados original

O sistema introduz ordem, rastreabilidade e controlo sem interferir no fluxo criativo do DJ.

Bem-vindo ao MDJPRO

A solução de engenharia de software mais avanzata para a gestão inteligente de bibliotecas musicais no macOS.

O Teu Novo Padrão:

Concebido para DJs profissionais e colecionadores que exigem perfeição na organização dos seus metadados. O MDJPRO não apenas organiza; transforma o teu fluxo de trabalho através de algoritmos DNA e uma automatização sem precedentes.

Como usar este manual

Este manual foi concebido como um guia técnico e operacional, não como material promocional. Aqui encontrará explicações claras de cada módulo, regras de utilização, avisos críticos, fluxos de trabalho recomendados e boas práticas profissionais.

Recomenda-se a leitura integral do manual antes de utilizar funções avançadas, especialmente aquelas que interagem com estruturas de pastas e metadados.

6. Biblioteca e Organização 🔒 PREMIUM ONLY

LIBRARY WIZARD (Gestor de Estrutura): (Utilização apenas com subscrições Premium) Módulo de automação que gera hierarquias de diretórios padronizadas no disco físico.

ESPECIFICAÇÕES DE PASTAS (DNA)

1
DÉCADAS (CRONOLÓGICO) Segrega o conteúdo por eras (anos 70, 80, 90). Ideal para sessões 'Open Format' onde a navegação temporal é prioritária.
2
INTERNATIONAL (REGIÕES) Organização por origem ou macro-regiões (Latim, Euro, Afro). Isola tendências culturais mantendo a independência regional.
3
POR GÉNEROS (TÉCNICO) Agrupa ficheiros por características sonoras fundamentais (House, Techno, Urban) independentemente da sua época ou procedência.
4
EVENTOS ESPECIAIS (MOBILE DJ) Concebido para eventos específicos (Casamentos, Corporativos) onde a funcionalidade operacional prima sobre o género musical.
5
O MEU LEGADO (HERANÇA) Repositório de longo prazo para material de alto valor (Rips de vinil, Masters). Prioriza a integridade do ficheiro sobre o acesso rápido.
6
PERFORMANCE SET Ambiente de baixa latência otimizado para rotinas, tone-play e competição. Contém apenas material verificado para palco.

13. Segurança e Boas Práticas – MDJPRO

A integridade da sua biblioteca musical é a nossa prioridade máxima. O MDJPRO implementa múltiplas camadas de segurança para garantir uma operação estável e livre de riscos.

🛡️ Protocolo Apple Sandbox

O MDJPRO opera sob isolamento seguro. O acesso a discos externos requer **autorização explícita** através de marcadores de segurança (SSB) persistentes.

🔑 Autorização Keychain (Biometria)

As suas credenciais SaaS são guardadas no Chaveiro encriptado do macOS. No primeiro arranque, o macOS solicitará **Touch ID ou Password** para criar o contexto de segurança silencioso.

💎 Estado 'BINGO' (Integridade)
  • ✔️ Permissões de escrita ativas.
  • ✔️ Caminho de origem validado fisicamente.
  • ✔️ Hash de integridade 1:1 confirmado.
👑

Regras de Ouro

Infraestrutura crítica recomendada para o MDJPRO Enterprise.

OBRIGATÓRIO
1
Hardware: Use cabos Thunderbolt oficiais ou certificados USB-IF para evitar quedas de tensão.
2
Energia: Mantenha a UPS (No-break) carregada para proteger a base de dados contra cortes elétricos.
3
Ambiente: Verifique a temperatura (Máx 30°C para discos NVMe) para prevenir o thermal throttling.
4
Protocolo: Nunca desligue volumes durante os processos de indexação ativa.
🛡️ PROTOCOL
**PROTOCOLOS DE SEGURANÇA MDJPRO:** O incumprimento das regras de ouro pode invalidar as garantias de recuperação de dados automatizada fornecidas pelo motor do MDJPRO.

14. Resolução de Problemas – MDJPRO

Guia técnico para a resolução de incidentes e terminologia padrão do ecossistema MDJPRO.

Guia de Resolução de Incidentes

🔴
Erro de Permissões: O acesso ao disco expirou por segurança do macOS.
➜ Ação: Execute a função **'Load Root'** novamente para renovar o marcador SSB.
📂
Ficheiros Perdidos (Missing): O caminho físico do ficheiro mudou fora do MDJPRO.
➜ Ação: Utilize a função **'Relocate'** no seu software DJ ou reimporte a pasta Master.
🎬
Falha de Vídeo: Codec não compatível com o motor nativo AVFoundation.
➜ Ação: Recodifique para **H.264/AAC** utilizando ferramentas padrão (ex. Handbrake).
🔑
Erro de Licença: Falha de comunicação com o servidor de validação.
➜ Ação: Verifique a sua ligação e reintroduza as suas credenciais no painel **Conta**.

Glossário Técnico

ASSET: Unidade individual de ficheiro de média (Áudio/Vídeo).

CRATE: Contentor lógico de organização específico do Serato DJ.

ID3 TAG: Padrão industrial de metadados incorporados em ficheiros.

ROOT: Diretório raiz ou ponto de montagem de um volume de dados.

SANDBOX: Mecanismo de segurança nativo do macOS para isolamento de processos.

XML: Linguagem de marcação usada para a troca de bases de dados.

🛡️ PROTOCOL
**PROTOCOLOS DE SUPORTE:** Em caso de comportamento errático persistente, reinicie a aplicação para limpar a cache temporária de processos da Apple.
