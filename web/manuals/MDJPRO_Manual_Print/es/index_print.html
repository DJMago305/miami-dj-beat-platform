Introducción General – MDJPRO

INFO NOTE
**AVISO IMPORTANTE:** MDJPRO trabaja directamente con archivos y estructuras reales del sistema. Aunque el software ha sido diseñado con protocolos de seguridad, **la responsabilidad final sobre el uso correcto recae en el usuario.** Se recomienda siempre trabajar con copias de seguridad.

Introducción

MDJPRO (Magic DJ Pro) es una plataforma profesional de gestión, auditoría y organización musical diseñada para DJs, productores y operadores de eventos que trabajan bajo estándares reales de la industria.

Desarrollada por Miami DJ Beat, MDJPRO no es un reproductor, ni un simple organizador de archivos. Es un sistema de control de librerías musicales creado para proteger, estructurar y optimizar uno de los activos más valiosos de un DJ profesional: su base de datos musical.

En un entorno donde los errores de organización, metadatos corruptos o escaneos incorrectos pueden comprometer una presentación en vivo, MDJPRO actúa como un firewall operativo entre el caos del disco duro y la estabilidad exigida por softwares como Serato DJ, Rekordbox and VirtualDJ.

Filosofía del Sistema

MDJPRO está construido bajo un principio fundamental:

"La música profesional no se improvisa. Se estructura, se valida y se protege."

Por esta razón, la aplicación opera mediante flujos de trabajo controlados, reglas claras y validaciones constantes. Cada acción dentro del sistema —desde cargar una carpeta raíz hasta editar un metadato— responde a una lógica diseñada para prevenir errores humanos, pérdida de datos y desorganización progresiva.

MDJPRO no sustituye al software de DJ. Lo prepara, lo alimenta correctamente y lo mantiene estable.

¿Para quién es MDJPRO?

  • DJs profesionales y residentes
  • DJs móviles y de eventos especiales
  • Productores y curadores musicales

Si tu música es parte de tu identidad profesional y de tu sustento, MDJPRO fue diseñado para ti.

Qué resuelve MDJPRO

  • Librerías desordenadas y difíciles de mantener
  • Metadatos incompletos, falsos o inconsistentes
  • Errores de escaneo en Serato u otros softwares
  • Pérdida de rutas al mover discos o carpetas
  • Falta de estructura para eventos, sets y décadas
  • Riesgo de dañar la base de datos original

El sistema introduce orden, trazabilidad y control sin interferir con el flujo creativo del DJ.

6. Librería y Organización 🔒 PREMIUM ONLY

LIBRARY WIZARD (Gestor de Estructura): (Uso solamente en subscripciones Premium) Módulo de automatización que genera jerarquías de directorios estandarizadas en el disco físico.

ESPECIFICACIONES DE CARPETAS (DNA)

1
DÉCADAS (CRONOLÓGICO) Segrega el contenido por eras (70s, 80s, 90s). Ideal para sesiones 'Open Format' donde la navegación temporal es prioritaria.
2
INTERNATIONAL (REGIONES) Organización por origen o macro-regiones (Latin, Euro, Afro). Aisla tendencias culturales manteniendo la independencia regional.
3
BY GENRES (TÉCNICO) Agrupa archivos por características sonoras fundamentales (House, Techno, Urban) independientemente de su época o procedencia.
4
SPECIAL EVENTS (MOBILE DJ) Diseñado para eventos específicos (Bodas, Corporativos) donde la funcionalidad operativa prima sobre el género musical.
5
MY LEGADO (HERENCIA) Repositorio de largo plazo para material de alto valor (Rips de Vinilo, Masters). Prioriza la integridad del archivo sobre el acceso rápido.
6
PERFORMANCE SET Entorno de baja latencia optimizado para rutinas, tone-play y competición. Contiene solo material verificado para 'Stage'.

13. Seguridad y Buenas Prácticas – MDJPRO

La integridad de su librería musical es nuestra máxima prioridad. MDJPRO implementa múltiples capas de seguridad para garantizar una operación estable y libre de riesgos.

🛡️ Protocolo Apple Sandbox

MDJPRO opera bajo aislamiento seguro. El acceso a discos externos requiere **autorización explícita** mediante marcadores de seguridad (SSB) persistentes.

🔑 Autorización Keychain (Biometría)

Tus credenciales SaaS se guardan en el Llavero cifrado de macOS. Al primer inicio, macOS solicitará **Touch ID o Password** para crear el contexto de seguridad silencioso.

💎 Estado 'BINGO' (Integridad)

  • ✔️ Ruta de Origen Validada físicamente.
  • ✔️ Hash de Integridad 1:1 confirmado.

14. Solución de Problemas – MDJPRO

Guía técnica para la resolución de incidentes y terminología estándar del ecosistema MDJPRO.

Guía de Resolución de Incidentes

🔴
Error de Permisos: El acceso al disco ha caducado por seguridad de macOS.
➜ Acción: Ejecute la función **'Load Root'** nuevamente para renovar el marcador SSB.
📂
Archivos Perdidos (Missing): La ruta física del archivo ha cambiado fuera de MDJPRO.
➜ Acción: Utilice la función **'Relocate'** en su software DJ o re-importe la carpeta Master.
🎬
Fallo de Video: Códec no compatible con el motor nativo de AVFoundation.
➜ Acción: Recodifique a **H.264/AAC** utilizando herramientas estándar (ej. Handbrake).
🔑
Error de Licencia: Fallo de comunicación con el servidor de validación.
➜ Acción: Verifique su conexión y reingrese sus credenciales en el panel **Cuenta**.

Glosario Técnico

ASSET: Unidad individual de archivo de medios (Audio/Video).

CRATE: Contenedor lógico de organización específico de Serato DJ.

ID3 TAG: Estándar industrial de metadatos incrustados en archivos.

ROOT: Directorio raíz o punto de montaje de un volumen de datos.

SANDBOX: Mecanismo de seguridad nativo de macOS para aislamiento de procesos.

XML: Lenguaje de marcado usado para el intercambio de bases de datos.

RULE PROTOCOL
**PROTOCOLOS DE SOPORTE:** En caso de comportamiento errático persistente, reinicie la aplicación para limpiar la caché temporal de procesos de Apple.
